## Deskripsi Project

Project ini merupakan implementasi praktikum Operasi Dasar Sinyal dan Citra menggunakan bahasa pemrograman Python.

Praktikum mencakup pengolahan citra digital dan sinyal audio dengan beberapa operasi dasar seperti:

- Membaca citra
- Operasi penjumlahan citra
- Operasi pengurangan citra
- Operasi perkalian citra
- Operasi pembagian citra
- Membaca sinyal audio
- Visualisasi waveform
- Penjumlahan amplitudo
- Delay sinyal
- Amplifikasi sinyal

---

## Library yang Digunakan

Project ini menggunakan beberapa library Python berikut.

- OpenCV (cv2)
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook

---

## Cara Menjalankan Notebook

1. Clone repository GitHub.

```bash
git clone https://github.com/username/operasi-dasar-sinyal-citra.git
```

2. Masuk ke folder project.

```bash
cd operasi-dasar-sinyal-citra
```

3. Install seluruh library.

```bash
pip install -r requirements.txt
```

4. Jalankan Jupyter Notebook.

```bash
jupyter notebook
```

5. Buka file

```
notebook/operasi_sinyal_citra.ipynb
```

6. Jalankan seluruh cell secara berurutan.
